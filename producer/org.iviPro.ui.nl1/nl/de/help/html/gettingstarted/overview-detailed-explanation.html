Überblick

Der SIVA Producer setzt sich aus verschiedenen Einzelkomponenten zusammen, die das Authoring eines interaktiven Videos ermöglichen. Mit Hilfe eines Videoschnitt-Tools können Szenen definiert werden. Ein weiteres Tool übernimmt das Erstellen und Anordnen des Szenengraphen. Außerdem wird ein Annotationseditor benötigt, der das Hinzufügen von Ressourcen zu bereits definierten Szenen ermöglicht. Um eine für den SIVA Player optimierte Version einer XML-Datei eines interaktiven Videos zu erhalten, wird vom SIVA Producer eine Exportfunktion bereitgestellt. Beim Export werden Player-relevante Daten in eine XML-Datei geschrieben.
