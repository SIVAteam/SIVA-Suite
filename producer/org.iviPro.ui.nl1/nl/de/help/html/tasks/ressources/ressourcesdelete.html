Löschen von Ressourcen

Um eine Ressource aus dem Medien-Repository zu löschen, wird mit der rechten Maustaste auf die Ressource geklickt. Im nun erscheinenden Menü muss nun der Eintrag "Medien löschen" ausgewählt werden. Eine Ressource kann außerdem durch Markieren der Ressource und einen Klick auf den Löschen-Button gelöscht werden.
