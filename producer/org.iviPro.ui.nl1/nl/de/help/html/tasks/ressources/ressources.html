Ressourcen

Um Ressourcen als Annotation zu einer Szene hinzufügen zu können, müssen diese zuerst in das Medien-Repository geladen werden. Von dort aus können sie via Drag and Drop auf eine Szene im Szenengraphen gezogen und so der Szene zugeordnet werden.
