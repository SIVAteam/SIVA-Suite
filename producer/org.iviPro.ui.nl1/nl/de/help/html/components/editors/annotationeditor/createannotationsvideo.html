Erstellen von Video-Annotationen

Eine Video-Annotation kann nicht bearbeitet werden. Sie kann nachdem sie ins Medien-Repository geladen wurde via Drag and Drop auf eine Szene im Szenengraphen gezogen werden. Danach können Start- und Endzeitpunkt der Wiedergabe festgelegt werden. Außerdem kann die Annotation mit einem Namen versehen werden.



Anzeigebereich

Mit einem Doppelklick auf den Anzeigebereich öffnet sich ein neues Fenster in dem man die Position und die Größe der Videoannotation bestimmen kann.



Mehr dazu siehe in der Hilfe Tasks > Annotationen > Erstellen von Annotationen
