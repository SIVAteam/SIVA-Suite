Erstellen von PDF-Annotationen

Wurde eine neue PDF-Annotation erstellt, so kann ein PDF-Dokument per Drag and Drop aus dem Medien-Repository in den Arbeitsbereich gezogen werden.


Nun kann eine Beschreibung des PDF-Inhalts, die fest mit der Annotation verbunden ist, erstellt werden.
