Editoren


Im SIVA Producer stehen verschiedene Editoren zur Verfügung. Diese dienen der Bearbeitung oder dem Erstellen von Szenen, Bildern, Texten oder des Szenengraphen.

Editoren im SIVA Producer sind:

  • ein Bildeditor
  • ein Szeneneditor
  • ein Annotationsedior
  • ein Szenengraph-Editor
  • ein Editor für das Inhaltsverzeichnis
