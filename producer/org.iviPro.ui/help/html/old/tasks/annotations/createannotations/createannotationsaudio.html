Erstellen von Audio-Annotationen

Eine Audio-Annotation kann nicht bearbeitet werden. Sie kann nachdem sie ins Medien-Repository geladen wurde via Drag and Drop auf eine Szene im Szenengraphen gezogen werden. Danach können Start- und Endzeitpunkt der Wiedergabe festgelegt werden. Außerdem kann die Annotation mit einem Namen benannt werden. Siehe dazu in der Hilfe Tasks > Annotationen > Erstellen von Annotationen
