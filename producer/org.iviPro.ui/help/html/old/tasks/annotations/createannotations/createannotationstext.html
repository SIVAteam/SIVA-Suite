Erstellen von Text-Annotationen

Text-Annotationen können, falls bereits erstellt, aus dem Medien-Repository per Drag and Drop auf eine Szene im Szenengraphen gezogen werden. Danach können Start- und Endzeitpunkt der Wiedergabe festgelegt werden. Außerdem kann die Annotation mit einem Namen benannt werden. Siehe dazu in der Hilfe Tasks > Annotationen > Erstellen von Annotationen

Eine Text-Annotation kann auch mit Hilfe des Text-Editors erstellt werden. Nach der Auswahl von "Neue TEXT-Annoation anlegen" öffnet sich der Text-Editor, in dem Text eingegeben werden kann.
