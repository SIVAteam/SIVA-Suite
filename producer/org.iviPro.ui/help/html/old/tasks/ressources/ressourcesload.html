Laden von Ressourcen

Um eine Ressource in das Medien-Repository zu laden, wird mit der rechten Maustaste auf das Medien-Repository geklickt. Im nun erscheinenden Menü muss nun der Eintrag "Medien laden" ausgewählt werden. Nach Navigation zur Datei und Auswahl der Datei wird diese durch einen Klick auf den Öffnen-Button ins Repository geladen. Eine Ressource kann außerdem durch einen Klick auf den Öffnen-Button und anschließende Auswahl der Datei geladen werden. Weiterhin ist es möglich die Datei aus dem Dateiexplorer des Betriebssystems via Drag and Drop ins Medien-Repository zu laden.
