Speichern eines interaktiven Video-Projektes

Beim Speichern des Projekts werden alle Informationen die beim erneuten Laden des Projekts nötig sind, um den gespeicherten Stand anzuzeigen, serialisiert in eine Datei geschrieben. Das Speichern wird durch Auswahl von "Projekt speichern" im Datei-Menü oder durch einen Klick auf den Speichern-Button gestartet.
