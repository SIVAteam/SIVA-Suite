A Simple Interactive Video Project

To create a simple interactive video project, only a few steps are necessary. First, a project has to be created. After that, a video has to be loaded. For this video, at least one scene has to be defined (maybe the whole video is a scene). This scene is added to the scene graph and linked with start and end node. After that, annotations can be added to the scene. After finishing the interactive video, it can be saved and/or exported for the SIVA Player.
