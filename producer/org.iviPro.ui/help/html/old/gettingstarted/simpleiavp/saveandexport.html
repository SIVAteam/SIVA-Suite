Save and Export

The following difference exists between saving and exporting an interactive video:

  • The export of the interactive video writes all information into a JSON or XML file which can be read by the player. Media files are put into a predefined folder structure and videos are converted into the output format. To start the export of a project, the export button is clicked.

  • Saving a project saves all information necessary to reload it into the SIVA Producer. To save a project, the save button is clicked.
