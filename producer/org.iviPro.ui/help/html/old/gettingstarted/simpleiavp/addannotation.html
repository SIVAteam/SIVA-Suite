Adding an Annotation

There are two different ways to add an annotation to a scene:

  • Drag & Drop: A file from the media repository is dragged and then dropped on the scene, the annotations should be shown with. After opening the annotation editor (by the button ), the display time of the annotation can be set, as well as a name.
  • Creating an annotation: Several ways exist to create and edit annotations after opening the annotation editor. Further information can be found in the help under Tasks > Annotations > Creation of Annotations.
