Tutorial - Creating a simple interactive video

This tutorial explains how to create a simple interactive video. Step by step it will make you familiar with the core components of the SIVA Producer and the workflow associated with the creation of hypervideos. The section is divided as follows:

  • Creating a video project
  • Adding media files
  • Cutting videos
  • Composing a scene graph
  • Adding scene annotations
  • Exporting a video project
