Adding scene annotations

The scene graph of your first interactive video with SIVA is now ready. Now you can add annotations to the scenes.

The following types of media can be used and chosen:

  • video
  • audio
  • picture
  • text
  • subtitle
  • PDF

For that purpose you have already loaded a picture into the media repository. This picture will be added as an annotation to a scene. Now move the picture via drag an drop into the scene it should be added to. A small red star appears in the scene symbol. It shows that an annotation already has been added to this scene. In the picture below this was marked by a green frame.



To edit an annotation click onto the button with the pen and the annotation editor will be opened.

An annotation always consists of content, duration and a name. According to the type of the annotation the content can be changed. The annotation can be given a name by the user. If no name is choses, this will be done by SIVA. For every annotation a starting and an ending point have to be set. This can be done in different ways:

  • entering the starting and ending point into the input fields
  • shifting the starting and ending point on the timeline
  • setting by clicking onto the timeline: the cursor is positioned on a required point on the timeline and the left mouse button is clicked. The defining of the ending point is done in the same manner, but the right mouse button has to be clicked.

If all settings are done, the annotation should be saved by clicking the "Save"-button. Then the annotation now appears in the timeline section in the window above.


Now the scene graph is ready. In the next section the saving and exporting of the interactive video is shown. Therefore click onto the next menu item "Saving and exporting a video project" in the help index on the left side or click onto the following link: Saving and exporting a video project
