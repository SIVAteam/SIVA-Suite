Exporting a video project

Now your fist interactive video is ready to save and export.

  • If you want to save the video, click on the 'save'-button.
    This has a green border in the picture. Saving means, that all information are saved, so that you can open or edit this project again in the SIVA-producer.
  • If you want to export the video, click on the 'export'-button.
    This has a red border in the picture. Exporting means, that all information are saved in a file, so that you can watch the video in a media player.

If you clicked on the 'export'-button, a window is open, in this you can choose a profile, that you want to export (for example a audio or video file). Furthermore you choose the target directory, in that the file should be saved. Afterwards click on the 'finish'-button to start the export process.
Your first interactive video is ready now!
