Create Richtext-Annotations

A richtext-annotation can also be created with the richtext-editor. After having chosen "Create new RICHTEXT-Annoation" the richtext-editor, where you can insert and format text.



Functions of the richtext-editor

The following functions are available in the toolbar:
  • Cut out text
  • Copy text
  • Paste copied text
  • Here you can edit the font size and font type
  • With this functions you can make the font bold, italy or underlined.
  • Here you can format the text left-aligned, right-aligned, centred or full justified
  • Here you can choose the colour
  • With this function you can insert a picture
  • Here you can instert a link

Inserted pictures can be deleted via rightclick.
If you change the text in one scene where it is used, as a consequence this affects every scene that includes that annotation.

For further details see: "Create annotations"
