Views


In the SIVA Producer there are different views at your disposal. They list and display different elements. Editors can be opened and files can be loaded and deleted by using the views.


Views are:

  • a Media-repository
  • a Scene-repository
  • a information-section
  • an overview

The Media-repository is structured into various files. The main files are "Audio", "Text", "Pictures", "PDF", "Subtitle" and "Videos", loaded files will be sorted automatically into the concerning file.
