Schließen und Wiederherstellen von Views


Views können durch einen Klick auf das Schließen-Kreuz rechts neben dem View-Namen geschlossen werden.

Zum Öffnen eines Views wird auf den Button in der linken unteren Ecke "Show View as a fast view" geklickt. In dem nun geöffneten Fenster sind im Ordner "Other" alle Views des SIVA Producers aufgelistet.



Hier ist es möglich, alle Views die geöffnet werden sollen zu markieren. Durch einen Klick auf "OK" öffnen sich die Views und können dann frei im Arbeitsbereich angeordnet werden.
