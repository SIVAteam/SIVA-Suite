Überblick

Nach dem Start des SIVA Producers stehen dem Benutzer verschiedene Werkzeuge und Bearbeitungsmöglichkeiten zur Verfügung. Nach dem Öffnen des Programms muss der Benutzer ein neues interaktives Video anlegen, oder ein gespeichertes laden. Eine Ressource (Video, Bild, Datei, Text, Audio) kann in das Medien-Repository geladen werden. Im Repository vorhandene Ressourcen können angezeigt oder abgespielt werden. Ein als Ressource geladenes Video kann in Szenen geschnitten werden. Das Schneiden des Videos in Szenen erfolgt entweder mit einem automatischen Videoschnitt-Tool (dabei wird das Video in Shots aufgeteilt) oder durch den Benutzer manuell. Bereits definierte Szenen befinden sich im Szenenrepository, von dem aus sie per Drag and Drop in den Szenengraphen eingefügt werden können. Szenen, die sich im Szenengraphen befinden, können durch Kanten miteinander verbunden werden. So entsteht eine zeitliche Abfolge der Szenen. Wenn sich eine Szene im Szenengraphen befindet, kann sie mit bereits geladenen Ressourcen annotiert werden. Darüber hinaus können neue Ressourcen erstellt werden, die anschließend in Form einer Annotation zur Szene hinzugefügt werden. Für diese Annotationen kann festgelegt werden, wo sie im SIVA Player erscheinen sollen und zu welchen Zeitpunkten sie ein- und wieder ausgeblendet werden sollen. Der Benutzer kann außerdem ausgeführte Aktionen rückgängig machen, das interaktive Video speichern oder für den SIVA Player exportieren.
