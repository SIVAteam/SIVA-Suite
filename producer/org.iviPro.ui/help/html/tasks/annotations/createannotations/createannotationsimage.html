Erstellen von Bild-Annotationen

Bild-Annotationen können aus dem Medien-Repository per Drag and Drop auf eine Szene im Szenengraphen gezogen werden. Danach können Start- und Endzeitpunkt der Wiedergabe festgelegt werden. Außerdem kann die Annotation mit einem Namen benannt werden. Siehe dazu in der Hilfe Tasks > Annotationen > Erstellen von Annotationen
