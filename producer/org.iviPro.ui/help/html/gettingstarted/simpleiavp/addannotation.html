Hinzufügen einer Annotation

Es bestehen zwei Möglichkeiten, um eine Annotation zu einer Szene hinzuzufügen:

  • Drag & Drop: Dazu wird eine ins Medien-Repository geladene Datei auf die Szene im Szenengraphen gezogen, bei der die Annotation angezeigt werden soll. Nach dem Öffnen des Annotationsediors (durch den Button ) kann die Anzeigedauer der Annotation eingestellt, sowie ein Name vergeben werden.
  • Erstellen einer Annotation: Nach dem Öffnen des Annotationseditors bestehen Möglichkeiten zum Anlegen und Editieren von Annotationen. Nähere Informationen zu den einzelnen Editoren und ihren Funktionalitäten sind in der Hilfe unter Tasks > Annotationen > Erstellen von Annotationen zu finden.
