Load a Video

Loading a video into the SIVA Producer can be accomplished the following way. Clicking on loads a file explorer to select a medium (video, image, audio...). After selecting a video, it will be shown in the media repository. To create an interactive video, at least one video has to be loaded.

List of supported video formats

  • .avi
  • .mp4
  • .mpg/.mpeg
  • .mov
  • .ogg
  • .wmv
