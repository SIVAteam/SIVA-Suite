Cutting a Video

In order to split a video into scenes, the video is selected in the media repository. This is done by a click with the right mouse button on the video and then selecting "Define Scenes".



Then a cutting tool opens and scenes can be created. The first scene is created automatically. Other scenes have to be created by clicking the "+" button on top of the window. The start and end time, the start and end frame, the name of the scene, and the keywords can be set in the lower area of the window. Furthermore, the start and the end time of a scene can be set as described in Tasks > Cutting a Video > Manual Video Editing
