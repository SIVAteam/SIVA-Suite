Overview

The SIVA Producer consists of different components for the authoring of an interactive video. Scenes can be defined with the help of the video cutting tool. Another tool is used for the creation and arrangement of the scene graph. Furthermore, an annotation editor is necessary which enables the adding of resources to already defined scenes. The export function is used to write the structure of the video into JSON and XML files. It also converts the media files and puts them into a predefined folder structure.
