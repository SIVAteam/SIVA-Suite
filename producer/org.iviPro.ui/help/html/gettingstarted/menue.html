Menu Structure

The menu structure is divided into the following areas: File, Edit, Project and Help.

The file menu is structured as follows:

  • New Project:
    Create a new interactive video project.
  • Open Project:
    Load an already created and saved interactive video project.
  • Save Project:
    Save an interactive video project for further processing.
  • Save Project as:
    Save an interactive video project for further processing under a different name.
  • Load Media Files:
    Load a medium (image, video, audio...) into the media repository for further processing.
  • Export:
    Export of an interactive video project for the SIVA Player.
  • Hand over Project:
    Export the project with all media files into a folder.
  • Close Project:
    Close the interactive video project without closing the SIVA Producer.
  • Exit:
    Exit the SIVA Producer.



The edit menu is organized as follows:

  • Undo:
    Undos the last performed action.
  • Redo:
    Redos the last undone action.
  • Scene graph:
    Opens the scene graph editor for editing the scene graph.
  • Reset Perspective:
    Reorganizes the editors and views in a way defined in the program.



The project menu is organized as follows:

  • Global Annotations:
    Opens the editor for the creation and editing of global annotations.
  • Table of Contents Editor:
    Opens the editor for the creation and editing of the table of contents for an interactive video.
  • Settings:
    Opens the settings editor for the player settings.



The help menu is structured as follows:

  • Help Contents:
    Opens the help
  • Search:
    Opens a search function for the help
  • About SIVA Producer:
    Information about the SIVA Producer
