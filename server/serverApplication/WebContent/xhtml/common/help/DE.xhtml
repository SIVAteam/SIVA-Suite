Startseite

Geben Sie den Videoschlüssel, den Sie erhalten haben in das Feld "Videoschlüssel", um direkt zum jeweiligen Video zu gelangen.
Nutzen Sie die Navigation im Kopfbereich zur Navigation durch die Website und zur Nutzung deren Fuktionen.

Registrieren

Erstellen Sie ein neues Benutzerkonto, um erweiterten Zugriff auf Gruppen und Videos zu erhalten.
Die folgenden Informationen sind dafür erforderlich:

  • eine gültige E-Mail-Adresse
  • ein sicheres Passwort (eine Buchstaben-/Zahlen-/Zeichenkombination)
    Dieses muss mindestens 3 der folgenden vier Zeichenkategorien enthalten:
    • Kleinbuchstaben
    • Großbuchstaben
    • Zahlen
    • Sonderzeichen
  • Ihr akademischer Titel
  • Ihr Vorname
  • Ihr Nachname
  • Ihr Geschlecht
  • Ihr Land

Ihr Vorname, Nachname, Land und Geschlecht wird dann in Ihrem Profil veröffentlicht. Alle weiteren Angaben sind optional und können wahlweise für das öffentliche Profil aktiviert oder deaktiviert werden.

Login

Sie können sich hier mit Ihrer E-Mail-Adresse und Ihrem Passwort einloggen und so zum Dashboard und erweiterten Funktionen gelangen.
Bevor Sie sich einloggen können, müssen Sie sich registrieren.

Passwort vergessen

Haben Sie Ihr Passwort vergessen, erhalten Sie über diese Funktion ein neues Passwort zu setzen. Dazu klicken Sie auf "Vergessen?" und geben die E-Mail-Adresse ein mit der Sie sich registriert haben. Das System schickt Ihnen dann eine E-Mail in der sich ein Link befindet. Folgen Sie diesem Link. Sie werden auf eine Seite verwiesen, auf der Sie Ihr neues Passwort zweimal eingeben müssen.

Hinweis: Dieser Link ist nur zwei Tage gültig.

Profil bearbeiten

Die Profil-Bearbeiten-Seite bietet Ihnen die Möglichkeit, Ihre Benutzerdaten zu ändern.
Obligatorische Benutzerdaten sind:

  • Ihre E-Mail-Adresse
  • Ihr sicheres Passwort (mindestens 7 Zeichen lang und eine Buchstaben-/Zahlen-/Zeichenkombination)
    Dieses muss mindestens 3 der folgenden vier Zeichenkategorien enthalten:
    • Kleinbuchstaben
    • Großbuchstaben
    • Zahlen
    • Sonderzeichen
  • Ihr akademischer Titel
  • Ihr Vorname
  • Ihr Nachname
  • Ihr Geschlecht
  • Ihr Land

Ihr Vorname, Nachname, Land und Geschlecht wird dann in Ihrem Profil veröffentlicht. Alle weiteren Angaben sind optional und können wahlweise für das öffentliche Profil aktiviert oder deaktiviert werden.

Dashboard

Das Dashboard bietet Ihnen die Möglichkeit auf erweiterte Funktionalitäten zuzugreifen.

  • Sie können Ihr Profil bearbeiten.
  • Sie erhalten eine Liste aller verfügbaren Gruppen. Bei diesen können Sie sich an- oder abmelden.
  • Autorisierte Dozenten oder Administratoren erhalten zusätzlich die Möglichkeit, Gruppen anzulegen, zu bearbeiten und zu löschen. Weiterhin können diese Benutzergruppen Benutzer als Teilnehmer hinzufügen oder von entfernen. Für Gruppen können Videos angelegt, bearbeitet oder gelöscht werden.
  • Autorisierte Administratoren können zusätzlich auf die Benutzerliste zugreifen und darüber Benutzer erstellen, bearbeiten, sperren und löschen. In der Benutzerliste kann ebenfalls nach bestimmten Benutzern gesucht werden.

Produktgruppen verwalten

Über die Liste der Produktgruppen können Sie sich zu bei einer Gruppe anmelden oder sich von einer solchen abmelden.
Mit Hilfe der Reiter können Sie die Liste weiter einschränken. Sie können dabei wählen, ob Sie alle Gruppen oder nur Gruppen bei denen Sie Teilnehmer oder Besitzer sind.

Bei Gruppe anmelden

Dieser Button ermöglicht es Ihnen, sich bei der jeweiligen Gruppe anzumelden. Sind Sie bei einer Gruppe angemeldet, erhalten Sie Zugriff auf Videos, die der Dozent dieser Gruppe ausschließlich für Teilnehmer freischaltet.

Von Gruppe abmelden

Sie können sich von einer Gruppe abmelden, wenn Sie an dieser nicht mehr teilnehmen und deshalb nicht mehr auf die Videos dieser Gruppe zugreifen müssen.

Gruppe anlegen

Zum Anlegen eines Videos benötigen Sie zunächst eine Gruppe, welcher diese dann zugeordnet werden kann.
Dazu sind folgende Eingaben erforderlich:

  • der Gruppenname
  • die Sichtbarkeit der Gruppe (Ist die Gruppe für jedermann sichtbar oder nur für Teilnehmer?)
Benutzer hinzufügen/entfernen

Zum Hinzufügen bzw. Entfernen von Benutzer zu bzw. von einer Gruppe stehen Ihnen entsprechende Buttons zur Verfügung.
Nutzen Sie die Suche, um entsprechende Benutzer schneller zu finden.

Dozenten hinzufügen/entfernen

Zum Hinzufügen bzw. Entfernen weiterer Dozenten zu bzw. von einer Gruppe stehen Ihnen entsprechende Buttons zur Verfügung.
Nutzen Sie die Suche, um entsprechende Benutzer schneller zu finden.

Gruppe bearbeiten

Die folgenden Daten können bearbeitet werden:

  • der Gruppenname
  • die Sichtbarkeit der Gruppe (Ist die Gruppe für jedermann sichtbar oder nur für Teilnehmer?)

Lesen Sie hier, wie Sie Benutzer in die Teilnehmerliste eintragen bzw. davon entfernen.

Gruppe löschen

Mit Hilfe dieses Buttons können Sie eine Gruppe löschen.

Hinweis: Alle mit der Gruppe verknüpften Daten gehen beim Löschen dieser verloren. Dies umfasst auch alle darin enthaltenen Videos.

Videos verwalten

Hier finden Sie eine Liste aller verfügbaren Videos sowie der für Sie verfügbaren Optionen.
Zusätzlich bereitgestellte Informationen sind die zugehörige Gruppe und der aktuelle Status des Videos.
Zum Sortieren klicken Sie auf die entsprechenden Pfeile.

Video anlegen

Zum Anlegen eines neuen Videos können die folgenden Informationen angegeben werden:

  • der Videoname
  • die Gruppe zur dieses Video gehört
  • eine optionale Beschreibung
  • die Veröffentlichungsart für das Video

Video bearbeiten

Folgende Videodaten können bearbeitet werden:

  • der Videoname
  • die Gruppe zu der das Video gehört
  • die optionale Beschreibung

Video konvertieren

Folgende Videoformate können konvertiert werden:

  • mp4
  • wmv
  • optional

Veröffentlichungseinstellungen bearbeiten

Zur Veröffentlichung eines Videos stehen die folgenden Möglichkeiten zur Verfügung:

  • das Setzen eines optionalen Aktivierungszeitpunkts
  • das Setzen eines optionalen Deaktivierungszeitpunkts
  • das Definieren eines Passworts zum Beantworten des Videos, sofern diese Veröffentlichungsart festgelegt wurde
  • das Generieren einer Liste von Videoschlüsseln, sofern diese Veröffentlichungsart festgelegt wurde
  • das Senden von Videoschlüsseln an Benutzer, sofern diese Veröffentlichungsart festgelegt wurde
Benutzer einem Video hinzufügen

Auf dieser Seite wählen Sie Benutzer aus, die das Projeckt betreten dürfen.
Um einen Benutzer hinzuzufügen, klicken Sie einfach auf "Benutzer zu Video hinzufügen". Um einen Benutzer wieder zu entfernen, klicken Sie auf "Benutzer von Video entfernen".
Über Suche ist es möglich Benutzer zu suchen.
Ist der Video zu einem Video durch Schlüssel geregelt, haben Sie ferner die Möglichkeit Videoschlüssel an eine Liste mit E-Mail-Adressen zu senden.
Dazu tragen Sie zeilenweise E-Mail-Adressen in das dafür vorgesehene Feld ein und klicken auf "Videoschlüssel an Liste senden".

Videoschlüssel

Hier können Sie die Anzahl der zu generierenden Schlüssel angeben und sich diese in Form einer ausdruckbaren Liste generieren lassen.

Video löschen

Hier können Sie das jeweilige Video löschen

Hinweis: Alle mit dem Video verknüpften Daten werden unwiderruflich gelöscht.

Benutzer verwalten

Die Benutzerliste kann mit Hilfe der Registerkarten eingeschränkt werden. Mögliche Einschränkungen sind die ausschließliche Auflistung von Dozenten oder Administratoren.
Neben dem jeweiligen Benutzer finden Sie Optionen zum Sperren/Entsperren, Bearbeiten oder Löschen des Benutzers.
Das Suchformular ermöglicht es Ihnen, einen bestimmten Benutzer zu suchen. Darüber befindet sich ein Button zum Anlegen eines neuen Benutzers.

Benutzer anlegen

Hier haben Administratoren die Möglichkeit, neue Benutzer, Dozenten oder Administratoren anzulegen. Erforderliche Daten dafür sind:

  • eine gültige E-Mail-Adresse
  • der akademische Titel
  • der Vorname
  • der Nachname
  • das Geschlecht
  • das Land
  • der Benutzertyp

Hinweis: Dem neune Benutzer wird ein für ihn generiertes Passwort an die angegebene E-Mail-Adresse geschickt.

Benutzer suchen

Sie können nach dem Vornamen, Nachnamen oder der E-Mail-Adresse des Benutzers suchen.

Hinweis: Sie müssen einen vollständigen Vornamen, Nachnamen oder eine vollständige E-Mail-Adresse für die Suche eines bestimmten Benutzers angeben.

Benutzer sperren

Das Sperren eines Benutzers hat zur Folge, dass sich dieser nicht mehr einloggen und somit nicht mehr die Funktionen registrierter Benutzer nutzen kann.

Benutzer entsperren

Das Entsperren eines Benutzers ermöglicht es diesem, sich wieder einzuloggen.

Benutzer bearbeiten

Mit Hilfe dieser Seite können Sie die Benutzerdaten bearbeiten
Folgende Daten sind obligatorisch:

  • die E-Mail-Adresse
  • der akademische Titel
  • der Vorname
  • der Nachname
  • das Geschlecht
  • das Land
  • der Benutzertyp


Ferner kann das Passwort des Benutzers zurückgesetzt werden. Dabei wird ein neues zufälliges Passwort generiert und diesem per E-Mail zugesendet.

Benutzer löschen

Hier können Sie den jeweiligen Benutzer löschen.

Hinweis: Das Löschen eines Dozenten hat zur Folge, dass alle Verknüpfungen desselben mit seinen Gruppen und seinen Videos werden. Ist bei einer Gruppe kein weiterer Dozent als Besitzer hinterlegt, wird diese Gruppe als unsichtbar markiert. Eine solche Gruppe kann von einem Administrator mit einem oder mehreren Dozenten versehen werden.

Kontakt

Hier haben Sie die Möglichkeit, Nachrichten, Verbesserungsvorschläge und Fehler an den Systemadministrator zu senden.
