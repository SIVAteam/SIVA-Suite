Home Page

Put the token you received in the field called "Video Token" to get direct access to the assigned video.
Use the menu on top to navigate through the provided functionality.

Register

Create a new account to get further access to groups, and videos.
You have to provide the following information:

  • a valid e-mail address
  • a safe password (letter-figure-symbol combination)
    Has to contain at least three of the following character sets:
    • lower-case letters
    • upper-case letters
    • figures
    • special characters
  • your academic title
  • your first name
  • your last name
  • your gender
  • your country

Your first name, last name, country and gender are published on your profile page. Further data can be entered optionally. You can set whether this optional data should be shown in your public profile.

Login Page

Login here with your e-mail address and password to get access to the dashboard and additional functions.
Before you can login you have to register.

Forgotten password

If you have forgotten your password, use this function set a new one. Simply click on "Forgotten?" and enter                     your e-mail address. The system will send you an email with a special link inside. Following this                     link you will be redirected to a page where you can enter your new password.

Note: This link is only two days valid.

Edit Profile

The profile page offers you to change your personal details.
The required details:

  • your e-mail address
  • your password (at least 7 characters, letter-figure-symbol combination)
    Has to contain at least three of the following character sets:
    • lower-case letters
    • upper-case letters
    • figures
    • special characters
  • your academic title
  • your first name
  • your last name
  • your gender
  • your country

Your first name, last name, country and gender are published on your profile page. Further data can be entered optionally. You can set whether this optional data should be shown in your public profile.

Dashboard

The dashboard provides the entry point to additional functions for logged in users.

  • Authorized get the possibility to edit their profile.
  • Authorized participants get a list of all available groups which they can sign up for or sign off from.
  • Authorized tutors additionally get the opportunity to create, edit and delete groups as well as an option to add and remove registered users from the list of participants. Within those groups videos can be created, edited and deleted.
  • Authorized administrators additionally get the opportunity to see a list of the registered users as well as an option to create tutors or administrators, ban, edit or delete users. A search function is also provided.

Manage Groups

Here you get a list of all the available groups that you can sign up for. You have tabs to order the groups.
You can display all visible groups, the groups you are attendant off or groups which you do own.

Sign Up For Group

Using this button you can sign up for an group. If you are signed up, you will get access to videos a tutor releases within this group.

Sign Off From Group

If you are no longer interested in the videos a certain group provides, you can sign off from the group using this button.

Create Group

To create a video you first have to set up an group to which it can be assigned after.
To create an group you have to provide:

  • a group name
  • an annotation whether the group should be only visible for attendants or not
Add/Remove Registered User

To add/remove a user to/from your group click the button "add user to group"/"remove user from group" within the row of the selected user - navigate through the lists pages by clicking the page-numbers.
Using the search box simplifies the process of adding users.

Add/Remove Tutor

To add/remove a tutor to/from your group click the button "add tutor to group"/"remove tutor from group" within the row of the selected tutor - navigate through the lists pages by clicking the page-numbers.
Using the search box simplifies the process of adding users.

Edit Group

The following information about your group can be changed:

  • the group name
  • the annotation whether the group should be only visible for attendants or not

Read here how to add/remove users to/from your group.

Delete Group

Using this button you will delete an group.

Note: All corresponding data gets lost! This includes videos.

Manage Videos

List all available videos with their related actions.
The list additionally contains all the relevant information such as the corresponding group and the current status of the video.
There are arrows to order the videos.

Create Video

To create a new video the following information is needed:

  • a video name
  • the group to which the video belongs
  • an optional description
  • the publication settings

Edit Video

The following video data can be edited:

  • the video name
  • the group to which the video belongs
  • the optional description

Video konvertieren

Folgende Videoformate können konvertiert werden:

  • mp4
  • wmv
  • optional

Edit Publication Settings

To distribute a previously created videos make use of one of the following possibilities:

  • Set the optional activation time for the video.
  • Set the optional deactivation time for the video.
  • Define a general password that allows to enter the video if this publication setting was set.
  • Generate a list of tokens if this publication setting was set.
  • Send token to specific registered users if this publication setting was set.
Add Users to Video

You can use this page to invite users to enter a video. Use the button "add user to video" to invite a user. To remove an already invited user use the "remove user from video" button. You may use the search field to easily locate a user you want to invite.
Instead of inviting registered users, you can also invite users by entering a list of e-mail addresses. Each e-mail address has to be entered in a seperate line into the field labeled "send tokens to list".

Access Tokens

Here you can set the number of generated tokens and generate a printable list of tokens.

Delete Video

To delete a video click on the "delete" button on the right.

Note: All corresponding data is getting lost!

Manage Users

The list of users can be ordered with tabs. You can show all users, only tutors or only administartors.
You can see a list of all registered users with shortcuts to ban/unban, edit or delete a user.
Using the field on top allows you to search for an existing user, clicking on "Create User" lets you create a new tutor or administrator.

Create User

Create a new tutor or administrator within the application.
At first you have to select whether you want to create a tutor or an administrator, then you have to enter the following additionally required details of the new user:

  • his/her (valid) e-mail address
  • his/her academic title
  • his/her first name
  • his/her last name
  • his/her gender
  • your country


You can also create a normal user in the same way explained above.

Note: The user gets a new generated password sent by email after creating him.

Search Registered User

Searching for a user providing an e-mail address or name.

Note: You have to enter a complete firstname, lastname or email-adress to find the searched user.

Ban user

Banning a user prohibits him to login any longer hence he gets locked out of the area for registered users.

Unban user

Unban the specified user to enable him to login again.

Edit User

This page offers you to change a users information.
The information that can be changed is:

  • his e-mail address
  • his academic title
  • his first name
  • his last name
  • his gender
  • your country
  • his role


In addition it is possible to reset his password to a new one which gets randomly generated and is sent to him in an e-mail.

Delete User

This button deletes a user from the system.

Note: Deleting a tutor results in removing him from all his groups and videos - should there be no assigned tutors left, the group is set to "invisible". To bring this group back online an administrator has to assign a tutor to it.
